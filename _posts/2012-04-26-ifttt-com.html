I've been using ifttt.com (which stands for If This Then That) for the past seven months. It's awesome and easy to use. I really love the clean, simple, minimalist design of the site.

Here's how it works. You tell it what "channels" you use. For this example... Wordpress and Twitter. Then plug in your user info for those sites. Next create a "task" by telling it... If I create a new post in Wordpress, then tweet about it. Done! Now every time a new post is created in Wordpress, it automatically tweets.

These automatic actions can be set up for Facebook, Tumblr, Instagram, Gmail, Flickr, YouTube, Soundcloud, Blogger (aka Blogspot), LinkedIn, Dropbox, Evernote, Posterous, Google Reader, Foursquare, Craigslist, Last.fm, Vimeo, and many more.

There are also "recipes", which are simply tasks that other people have created and made public. One recipe that I'm using is: If someone follows me on Twitter, then follow them on Twitter.
