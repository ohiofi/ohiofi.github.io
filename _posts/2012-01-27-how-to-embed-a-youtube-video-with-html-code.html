Video and audio were not originally part of HTML code. Instead, it was necessary to use work-arounds that use Adobe Flash. HTML5 (which is still being developed and currently does not work on all web browsers) will be the first version of HTML that will use <video> and <audio> code, and will not need to rely on Flash. Because the web is still transitioning to HTML5, we will look at the work-around for embedding a YouTube video with HTML code.

1.) Click the Share button beneath a YouTube video.

2.) Click the Embed button.

3.) Copy the "iframe" code OR put a check next to Use Old Embed Code and copy the "object" code.

4.) Click the Edit HTML tab in Blogger.

5.) Paste the code.
